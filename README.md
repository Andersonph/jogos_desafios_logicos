# O Pitch – A Profecia do Jogo

Nossa ideia inicial é desenvolver um jogo digital focado em desafios de lógica e raciocínio, no qual o jogador deverá resolver diferentes enigmas para avançar de fase. 

A proposta busca unir entretenimento e aprendizado, estimulando o pensamento crítico, a análise de problemas e a tomada de decisões.

O objetivo principal é criar uma experiência divertida e desafiadora, capaz de incentivar o desenvolvimento do raciocínio lógico por meio de mecânicas simples e intuitivas.

# **Essência do Jogo**

**Gênero**: Estratégia / Raciocínio Lógico
O jogo será baseado na resolução de desafios e problemas lógicos, exigindo que o jogador analise situações, identifique padrões e encontre soluções para avançar

**Mecânica Central**

O jogador interagirá com elementos presentes em cada desafio utilizando controles simples por meio de cliques do mouse. 
Cada fase apresentará uma situação específica que deverá ser resolvida respeitando determinadas regras. O jogador deverá interpretar as informações disponíveis, testar estratégias e encontrar a solução correta.

**Objetivo do Jogador**

Resolver os desafios propostos para avançar pelas fases do jogo.
A idéia é que cada desafio concluído desbloqueia o próximo, incentivando a progressão contínua até a conclusão da aventura.

**Progressão**

O jogo contará inicialmente com:

- 8 desafios independentes;
- Sistema de fases;
- Desbloqueio progressivo;
- Contador de tentativas ou movimentos;
- Sistema de pontuação baseado no desempenho do jogador.

A ideia inicial é que cada integrante do grupo desenvolva um desafio diferente, permitindo a participação ativa de todos no projeto. Caso identifiquemos que o game fique pesado para rodar, poderemos aplicar versões mais leves dos desafios sem comprometer a proposta principal.

# **O Mundo**

O jogo será composto por diferentes desafios de lógica, onde cada fase apresentará um novo problema a ser resolvido. O jogador será o protagonista da aventura e deverá utilizar raciocínio lógico, estratégia e observação para superar os obstáculos e avançar pelas fases. 

Um pouco diferente dos jogos tradicionais de ação, o principal desafio não será enfrentar inimigos, mas sim analisar situações, interpretar regras e tomar decisões corretas diante de problemas cada vez mais complexos, proporcionando uma experiência de aprendizado, evolução e conquista.

# Experiência do Jogador

O jogo terá uma proposta estratégica e reflexiva.
O foco não será velocidade ou reflexos, mas sim:

- Pensamento lógico;
- Planejamento;
- Observação;
- Resolução de problemas;
- Aplicação de conceitos aprendidos.

Além do entretenimento, alguns desafios poderão ser inspirados em exercícios e conteúdos já trabalhados em sala de aula, permitindo que o jogo também funcione como ferramenta de reforço do aprendizado.
A dificuldade aumentará gradativamente conforme o avanço das fases.

# Identidade Visual

O projeto utilizará uma interface limpa e intuitiva, facilitando a compreensão das regras e dos desafios.
A proposta visual inicial será composta por:

- Elementos gráficos em estilo cartoon ou minimalista;
- Menus inspirados em jogos clássicos;
- Sistema de fases visual;
- Telas de vitória e derrota;
- Indicadores de progresso e pontuação.

**Menu Principal**

O jogo contará com uma tela inicial semelhante aos menus tradicionais dos videogames, contendo opções como:

- Iniciar Jogo;
- Seleção de Fases (desbloqueadas);
- Pontuação;
- Créditos;
- Sair.

Essa estrutura permitirá melhor organização e navegação para o usuário.

- Sair.

# Objetivo Final

Em vez de focar em combate ou ação, a proposta desafia o jogador a pensar, analisar e construir soluções para diferentes problemas, promovendo uma experiência divertida e educativa ao mesmo tempo.
Além disso, a divisão dos desafios entre os integrantes da equipe permitirá que todos participem ativamente do desenvolvimento, contribuindo para a construção de um jogo mais variado e completo.
